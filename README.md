# Data Jobs Dashboard (Power BI)

## Introduction

This project focuses on analyzing the **data job market** using a real-world dataset of 2024 job postings.

The main goal was to bring scattered information like job roles, salaries, locations, and hiring platforms into a single dashboard that is easy to understand and explore.

This dashboard is useful for:
- Job seekers  
- Career switchers  
- Students exploring data roles

### Drill-Through Analysis (Job Role Deep Dive)

 Allows users to move from a high-level overview to a detailed analysis of a specific job role.

 <img width="2480" height="1390" alt="Screenshot 2026-04-22 225709" src="https://github.com/user-attachments/assets/6b68d1f0-26b1-4e8e-a8d7-216293bfbf9b" />

#### How it Works
- Users can right-click on any job title in the main dashboard  
- Select **Drill Through → Job Details Page**  
- This opens a dedicated page with in-depth insights for that role  

#### What It Shows
For each selected job role, the drill-through page provides:

- Salary distribution (yearly & hourly)  
- Work-from-home availability (WFH %)  
- Degree requirements  
- Health insurance availability  
- Global job distribution (map)  
- Top hiring platforms  

#### Why This is Important
This feature helps in:

- Understanding a specific role in detail  
- Comparing job roles beyond summary-level data  
- Making more informed career decisions
